Wordpress Party Hall Booking Management System.

== Description ==

"Party Hall Booking Manager" plugin will enable online booking services for your site.
you can show and manage your Community Center/Party houses (or something else) availability for rent.
You can manage the bookings (availability) on a daily basis. booking management and booking calendar page will be created automatically and shortcode to generate
calendar will be automatically copied. you can add/edit Community center or any service and add/edit or delete bookings from admin to manage the full system.

== Installation ==

- Copy/Upload 'ccb_scbooking' folder to the '/wp-content/plugins/' directory
- Activate 'Party Hall Booking Manager' form wp plugin option in admin area
- Plugin will appear in the menu bar of WP Dashboard
- create party hall/Community center/resort/else for booking.
- please save primary menu if no rooms gallery or booking calendar is showing.
